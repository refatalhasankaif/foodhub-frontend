# FoodHub Frontend

A modern food ordering platform frontend built with Next.js and Tailwind CSS. It delivers a fast, responsive, and intuitive experience for customers to browse meals, place orders, and manage their accounts, while enabling sellers and admins to operate efficiently through role-based dashboards.

## Tech Stack

- Next.js 16
- TypeScript
- Tailwind CSS
- Shadcn UI

---

## Features

- Modern food browsing experience with categorized meals.
- Role-based dashboards for Customer, Seller, and Admin.
- Secure authentication using Better Auth.
- Add to cart and seamless checkout flow.
- Seller panel for managing menu items and orders.
- Admin panel for platform control and user management.
- Fully responsive design across devices.
- Clean and scalable frontend architecture.
